# 🎓 EquidadeIA

> Plataforma de geração de atividades pedagógicas adaptadas para alunos
> neurodivergentes, usando Inteligência Artificial.

## 📌 Sobre o Projeto
Desenvolvido por alunos do CETI Letícia Macedo como parte do projeto
"Do Piauí para o Mundo" — 13ª GRE, Piauí, 2026.

## 🧠 Perfis Atendidos
- 🔵 TEA — Transtorno do Espectro Autista
- ⚡ TDAH — Déficit de Atenção e Hiperatividade
- 📖 Dislexia
- 🔢 Discalculia

## 🚀 Funcionalidades
- Login simulado de professores
- Gerador de atividades conectado à IA Claude (Anthropic)
- Catálogo com atividades adaptadas por perfil

## 🛠️ Tecnologias
HTML5 · CSS3 · JavaScript
